## Short description (132 chars max)

Tiny dino chaos for any webpage: a sprite dinosaur chomps visible text line by line, with optional herd stampedes.

## Detailed description

Dinosaur Eats is a playful browser extension that drops a tiny pixel dino onto the page to chomp through visible text.

Features:

- Single-dino and herd stampede modes
- Chomp audio with autoplay-safe behavior
- Toolbar ON/OFF toggle
- Repeat rampages for ongoing chaos

How it works:

- Runs a content script on pages you visit
- Finds visible text blocks and animates bite-by-bite removal
- Stores only one local preference: whether the extension is enabled

Privacy:

- No analytics
- No tracking
- No remote data collection
